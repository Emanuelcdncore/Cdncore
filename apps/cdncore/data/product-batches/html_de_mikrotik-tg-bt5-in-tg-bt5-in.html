Intelligente Ortung für Ihre Anlagen.

Der TG-BT5-IN ist ein schlankes und elegantes Bluetooth-Low-Energy-(BLE-)Ortungs-Tag von MikroTik, das für die Verfolgung von Anlagen in Innenräumen konzipiert ist. Perfekt für die Lagerlogistik oder die Ortung von Geräten auf Baustellen, überträgt dieses kleine Tag seinen Standort an Bluetooth-Gateways (wie das MikroTik KNOT) und ermöglicht so den Aufbau eines erschwinglichen und flexiblen Ortungs- und Telemetriesystems.

Warum das TG-BT5-IN-Tag wählen?

Einfache und effektive Näherungsverfolgung

Überwachen Sie Ihre wertvollen Objekte auf einfache Weise. Befestigen Sie das Tag einfach an dem Gegenstand, den Sie orten möchten, seien es Werkzeuge, Paletten oder elektrische Geräte, und das nächstgelegene BLE-Gateway erkennt seine Anwesenheit und sendet die Informationen an Ihren Server. Lassen Sie sich benachrichtigen, sobald Ihre Ladung im Lager eintrifft!

Langlebige Batterie

Die Tags verbrauchen so wenig Energie, dass ihre integrierte 220-mAh-Batterie jahrelang halten kann. Das Übertragungsintervall ist konfigurierbar; bei einem Intervall von 1 Sekunde hält die Batterie beispielsweise normalerweise bis zu zwei Jahre, und bei längeren Intervallen kann sie deutlich länger halten.

Integrierter Beschleunigungssensor

Zusätzlich zur Ortung enthält das Tag einen Beschleunigungssensor, der Bewegung, Neigung oder sogar einen Sturz des Objekts erkennen kann (Freifallerkennung) und so wertvolle Telemetriedaten über den Status Ihrer Anlagen liefert.

Kompatibilität mit mehreren Formaten

Standardmäßig überträgt das Tag Pakete im MikroTik-Telemetrieformat, es ist aber auch vollständig mit den Standards iBeacon und Eddystone kompatibel und gewährleistet so seine Integration in eine breite Palette von Anwendungen und Ortungssystemen von Drittanbietern.

Ideal für wen?

Lagerlogistik

Zur Verfolgung von Inventar, Paletten oder Geräten innerhalb eines Lagers. Wissen Sie immer, wo sich Ihre Anlagen befinden, und erhalten Sie automatische Benachrichtigungen, wenn sie bewegt werden.

Anlagenverwaltung auf Baustellen

Befestigen Sie Tags an wertvollen Werkzeugen und Geräten, um sie auf einer großen Baustelle mithilfe von über das gesamte Gelände verteilten KNOT-Gateways leicht zu lokalisieren.

IoT-Anwendungen (Internet der Dinge)

Für jedes IoT-Projekt, das eine Näherungs-, Bewegungs- oder Neigungserkennung erfordert, verwenden Sie ein Gerät mit geringem Stromverbrauch und langlebiger Batterie.
